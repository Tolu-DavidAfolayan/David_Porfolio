Case Study: Titanic Survival Prediction

This project applied machine learning to the classic Titanic dataset to predict passenger survival. The aim was to explore the relationship between socio-demographic factors, ticket class, and other attributes in determining survival likelihood.

Project Overview

The Titanic dataset is one of the most well-known datasets in data science, containing information about the passengers aboard the ship during its tragic voyage in 1912. The goal was to build models that predict whether a passenger survived based on features such as age, sex, class, family size, and fare.

Using Python (Pandas, Scikit-learn, Matplotlib), several models were trained and compared. The analysis provided insights into survival patterns and demonstrated how machine learning can extract meaningful signals from historical data.

The Challenge

Key challenges of the Titanic dataset included:

  • Handling missing values in Age and Cabin features.
  • Encoding categorical variables like Sex and Embarked.
  • Dealing with class imbalance (more passengers did not survive than survived).
  • Selecting the best-performing machine learning model for accuracy and generalization.

The Approach & Solution

The project pipeline included:

  • Data Cleaning → Filled missing values, dropped irrelevant columns, and engineered new features like FamilySize and IsAlone.
  • Exploratory Data Analysis → Explored survival rates by passenger class, gender, and age groups.
  • Feature Engineering → Converted categorical variables to numeric and scaled continuous features.
  • Model Development → Trained Logistic Regression, Decision Tree, Random Forest, and Support Vector Machine.
  • Evaluation → Compared models using accuracy, precision, recall, and F1-score.
